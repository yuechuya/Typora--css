# Typora 个性化主题 —— 水墨·星河

一个为 Typora 打造的精致主题，融合了中文字体、背景图片和舒适排版，让写作更具沉浸感。

---

## ✨ 主题特点

- 🖼️ **背景图片**：柔和的淡墨背景，可自定义替换
- 📝 **中文字体优先**：首选“楷体”，阅读更亲切
- 🎨 **优雅配色**：清爽的蓝、红、紫点缀，不刺眼
- 📊 **表格美化**：圆角边框 + 悬停效果，数据更清晰
- 💻 **代码块风格**：带“红绿灯”装饰，仿Mac终端
- 📏 **宽屏适配**：最大宽度1400px，阅读舒适
- 🌙 **沉浸式写作**：聚焦模式支持，减少干扰

---

## 📦 安装方法

1. 下载本仓库中的 `github.css` 文件（或您自定义的名称）。
2. 打开 Typora，进入 **偏好设置** → **外观** → **主题**。
3. 点击 **“打开主题文件夹”**。
4. 将下载的 `.css` 文件复制到该文件夹中。
5. 重启 Typora，在菜单栏 **“主题”** 中选择本主题即可。

## 📝 字体说明

主题优先使用本地“楷体”字体，如果系统没有该字体，则会降级到 `Open Sans` 及系统默认字体。
